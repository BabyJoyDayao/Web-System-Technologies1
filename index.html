Menu

second

  • Classic horizontal menu: the most common and well-recognized type of menu which presents the core navigation organized as a horizontal line in the website header.
  • Sidebar menu: a classic type, presents a vertical list of options sticking to the left or right side of the web page.
  • Dropdown menu: for content-heavy websites; additional options open below the primary one when clicked or hovered.
  • Megamenu: a large, expandable menu in a two-dimensional dropdown layout; effective for many options.
  • Hamburger menu: when the hamburger button (three lines) is clicked, the menu expands. Often used in mobile versions.
